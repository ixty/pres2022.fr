Aide et informations
contact@pres2022.fr Ce projet analyse les tweets recuillis via la streaming API de Twitter. Pour chaque candidat, deux mots clés sont envoyés à twitter: le nom d'utilisateur (screen_name) et le nom complet du candidat, tel que renseigné sur twitter (full_name). Le classement par jour - page "Rangs" - est établi avec le total du nombre de tweets, retweets, réponses et citations dont le contenu, l'auteur ou les entités mentionnées contiennent les mots clés du candidat. La page "graph" représente ce classement sous forme visuelle. La page "Tags" affiche des bulles plus ou moins grosses pour chaque candidat, en fonction du volume de tweet de la journée, ainsi que les hashtags et emojis les plus utilisés. Le tout est affiché sous forme de "force directed graph". La page "tweets" affiche pour chaque jour et chaque candidats, le top des tweets publiés par le candidat lui-même ainsi que les plus "likés", les plus "retweetés" et ceux publiés par les comptes ayant le plus d'abonnés (généralement les médias). Twitter n'offre aucune garantie quand à l'exhaustivité des données envoyées, ce projet ne le peut donc pas non plus. Code source disponible ici: https://github.com/ixty/chetane
